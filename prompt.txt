ខ្ញុំមាន Django loan website project ( Mana_system ) មួយ ហើយខ្ញុំចង់អោយ Claude ជួយ implement feature ២ យ៉ាង:

1. Custom Status Label Feature:
Staff អាច type custom text ក្នុង "Custom Status" box នៅ staff panel។ 
Text នោះនឹងបង្ហាញជំនួស default account_status នៅ dashboard/profile page របស់ user។
- model field ដែលត្រូវបន្ថែម: custom_status_label = models.CharField(max_length=120, blank=True, default="")
- ប្រើ migration create field នេះ
- staff_user_update view ត្រូវ save: u.custom_status_label = (request.POST.get("custom_status_label") or "").strip()
- dashboard_view ត្រូវ fetch fresh user ពី DB (User.objects.get(pk=request.user.pk)) ហើយ pass: status_display = (fresh_user.custom_status_label or "").strip() or (fresh_user.account_status or "").strip() or "ACTIVE" និង status_color = (fresh_user.account_status or "ACTIVE").strip().upper()
- realtime_state API ត្រូវ fetch fresh user ពី DB ហើយ return "custom_status_label": custom_label ក្នុង JSON

2. Correct Color Behavior (Green/Yellow/Red):

Color logic (ប្រើ account_status សម្រាប់ color, ប្រើ custom_status_label សម្រាប់ display text):
- GREEN: ACTIVE, APPROVED, LOAN_PAID, WITHDRAWAL_SUCCESSFUL, ACCOUNT_UPDATED
- YELLOW: APP_MAINTENANCE, INVALID_DETAIL, RENEW_DOCUMENT_REQUIRED  
- RED: Everything else (BANNED, FROZEN, LOCKED, etc.)
